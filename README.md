# Monthly Budget Tracker

אפליקציית מעקב תקציב חודשי אישי — בנויה עם React, TypeScript ו-Supabase.

## מה זה?

לוח בקרה פיננסי שמאפשר לך לעקוב אחר הכנסות והוצאות חודשיות, לצפות ביתרה המצטברת לאורך זמן, ולנתח את דפוסי ההוצאות שלך בצורה ויזואלית.

## תכונות עיקריות

- **הוספת עסקאות** — הכנסות והוצאות עם קטגוריות
- **לוח בקרה חודשי** — סיכום מיידי של הכנסות, הוצאות ויתרה
- **ניתוח חודשי** — גרפים ותרשימים של הוצאות לפי קטגוריה
- **יתרה מצטברת** — מעקב אחר הצמיחה הכספית לאורך חודשים
- **תמיכה בעברית ואנגלית** — ממשק דו-לשוני מלא
- **עיצוב Glassmorphism** — ממשק כהה ומודרני עם אפקט זכוכית
- **גיבוי נתונים** — ייצוא וייבוא נתונים
- **אימות ביומטרי** — תמיכה ב-Face ID / Touch ID

## טכנולוגיות

- **Frontend:** React 18 + TypeScript + Vite
- **UI:** Tailwind CSS + Radix UI
- **Backend:** Supabase (PostgreSQL + Auth)
- **Deploy:** Netlify
